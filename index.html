This website is about Grace Hopper. We are doing this website for a project in I101. The project is for us to create a website about an underrepresented innovator. We choose Grace Hopper to be our underrepresented innovator. She has made crucial innovations to the field of computer science. She developed the first IDE, the programming language COBOL, and many other contributions.
